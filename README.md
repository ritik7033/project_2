#  Dog Image Generator

This is a simple web project that fetches random dog images using the Dog CEO API.

##  Features
- Fetches random dog images
- Button with hover and animation effects
- Clean UI with gradient background
- Responsive image design

##  Technologies Used
- HTML
- CSS
- JavaScript
- Axios
- Dog CEO API

##  How It Works
1. Click the "Generate Dog" button  
2. A random dog image appears  
3. Enjoy cute dog pictures 

##  Project Structure
- index.html  
- style.css  
- app.js  
- README.md  

##  API Used
https://dog.ceo/api/breeds/image/random
